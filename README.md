# 🍅 pomoDORONG

> 음악 스트리밍과 함께하는 감성 뽀모도로 타이머

## 📖 소개

pomoDORONG은 YouTube 음악과 함께 집중력을 높여주는 뽀모도로 타이머입니다.

### 주요 특징

- ⏱️ **4가지 프리셋**: 클래식(25/5), 스탠다드(50/10), 딥워크(120/30), 커스텀
- 🎚️ **자유로운 시간 설정**: 1분~240분 작업, 1분~60분 휴식
- 🎵 **복수 음악 재생**: 작업용/휴식용 트랙 자동 재생
- 📼 **이어듣기**: 모드별 재생 위치 기억
- 🔊 **볼륨 조절**: 0-100% 세밀한 조절
- 📊 **세션 카운터**: 완료한 세션 수 추적
- 📈 **프로그레스 바**: 남은 시간 시각화
- 💾 **자동 저장**: 설정 및 재생 위치 localStorage 저장

## 🚀 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 빌드
```bash
npm run build
```

## 🛠 기술 스택

### Frontend

- **React 19** - UI 라이브러리
- **TypeScript** - 타입 안전성
- **Vite** - 빌드 도구
- **Zustand** - 상태 관리

### External APIs

- **YouTube IFrame API** - 음악 재생

### 배포

- **Vercel** - 자동 배포

## 🎯 사용 방법

### 1. 프리셋 선택
- 클래식, 스탠다드, 딥워크, 커스텀 중 선택
- 커스텀 선택 시 슬라이더로 시간 조절

### 2. 타이머 시작
- **시작** 버튼 클릭
- 음악 자동 재생
- 작업 시간 카운트다운

### 3. 음악 컨트롤
- **볼륨 조절**: 슬라이더로 조절
- **다음 곡**: 버튼으로 건너뛰기
- **재생목록**: 트랙 직접 선택

### 4. 모드 전환
- 작업 시간 종료 → 자동으로 휴식 모드
- 휴식 시간 종료 → 자동으로 작업 모드
- 음악도 자동 전환
- 이전 재생 위치 기억
