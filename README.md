# 🍅 pomoDORONG

> 음악 스트리밍과 함께하는 감성 뽀모도로 타이머

## 📖 소개

pomoDORONG은 YouTube 음악과 함께 집중력을 높여주는 뽀모도로 타이머입니다.

### 주요 특징

- ⏱️ **자유로운 시간 설정**: 25분부터 4시간까지, 나만의 작업 시간 설정
- 🎵 **음악 스트리밍**: YouTube 음악을 배경으로 재생
- 🔄 **자동 모드 전환**: 작업 시간과 휴식 시간 자동 전환
- 💾 **세션 카운팅**: 완료한 작업 세션 기록

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 빌드

```bash
npm run build
```

## 🛠 기술 스택

### Frontend

- **React 19** - UI 라이브러리
- **TypeScript** - 타입 안전성
- **Vite** - 빌드 도구
- **Zustand** - 상태 관리

### External APIs

- **YouTube IFrame API** - 음악 재생

### 배포

- **Vercel** - 자동 배포

## 🎯 사용 방법

1. **시작**: 시작 버튼을 클릭하여 타이머 시작
2. **일시정지**: 필요 시 일시정지 후 재개 가능
3. **리셋**: 타이머를 초기 상태로 되돌리기
4. **음악**: 타이머와 자동으로 연동되어 재생/일시정지
